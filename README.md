# Playwright QA Automation Project

This project contains UI and API automation tests using Playwright.

## Technologies
- Playwright
- TypeScript
- Node.js
- GitHub Actions (CI/CD)

## Test Scenarios
- Login test (positive & negative)
- Add product to cart
- API testing

## How to run tests

Install dependencies:

npm install

Run tests:

npx playwright test

## CI/CD

Tests run automatically on GitHub Actions.
